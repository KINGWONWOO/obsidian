##### 17강 : 연결된 복제

	- 상단의 Pivot Point 변경을 클릭
	![[Pasted image 20241222195956.png]]
---
- Linked Duplicate : 연결된 복제 시 한 Object의 변경이 다른 Object에도 적용됨
- 2개의 Object를 연결하고 싶으면
	1. 연결하고 싶은 Object 클릭
	2. Shift + 클릭으로 연결될(기준) Object 다중 클릭
	3. Ctrl + L 클릭 후 Linked Object 클릭
- 연결 해제 법 : 선택 후 Object -> Relation -> Single User -> Mesh & Data
