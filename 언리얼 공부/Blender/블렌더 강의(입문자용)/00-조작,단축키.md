| local view | numpad /      |                        |     |
| cursor 설정  | shift + S     |                        |     |
## 메쉬 조작
| 조작               | 단축키         | 추가설명                                                              |
| ---------------- | ----------- | ----------------------------------------------------------------- |
| 메쉬 생성 메뉴         | shift + A   |                                                                   |
| 3D 커서 이동         | shift + 우클릭 | 커서 위치에 Mesh 생성됨                                                   |
| 메쉬 이동            | G           | X/Y/Z키를 추가로 눌러 축에 고정 가능                                           |
| 메쉬 회전            | R           | X/Y/Z키를 추가로 눌러 축에 고정 가능                                           |
| 메쉬 스케일링          | S           | X/Y/Z키를 추가로 눌러 축에 고정 가능                                           |
|                  |             | 위의 모든 과정 우클릭으로 취소 가능, 숫자 입력으로 특정 거리로 이동. Shift + 축으로 그 축 제외만 변경가능 |
| 미세 이동            | G, shift    |                                                                   |
| 스냅핑 이동           | G, Ctrl     |                                                                   |
| 메쉬 속성값           | N           |                                                                   |
| 메쉬 복사            | shift + D   |                                                                   |
| 메쉬 이동 취소         | Alt + G     |                                                                   |
| 메쉬 회전 취소         | Alt + R     |                                                                   |
| 메쉬 스케일링 취소       | Alt + S     |                                                                   |
| Rename           | F2          |                                                                   |
| 새로운 컬렉션          | M           | 박스 drag 후 설정, 필요없는 Mesh는 ctrl 드래그로 제외시키기                          |
| 비례 편집 툴          | O           |                                                                   |
| Linked Duplicate | Alt + D     |                                                                   |
| Linked 메뉴 열기     | Ctrl + L    |                                                                   |
| 메쉬 값 적용          | Alt + A     |                                                                   |
| 메쉬 조인            | Ctrl + J    | 기준 물체를 마지막에 선택                                                    |
| 이전 동작 반복         | Shift + R   |                                                                   |
| link 된 것만 선택     | L           |                                                                   |
| 그 외 선택           | I           |                                                                   |
| parent 설정        | Ctrl + P    |                                                                   |
| cursor 조작        | shift + S   |                                                                   |

##  Modifier
- Modifier를 통해 Mesh에 다양한 변형 추가 가능
![[Pasted image 20241222192601.png]]
- 모든 Modifier는 Apply 하기 전까지는 Modeling 창에서만 적용되어 있음(다른 곳에서 편집하고 세부 조정하기 위해), Apply 시 되돌리기 불가
- Decimate Modifier를 통해 Mesh에 Polygon 수를 줄이기 가능
- Mirror Modifier를 통해 대칭 제작 가능. 이때, Clipping을 키면 대칭 접합부가 붙어서 동작
