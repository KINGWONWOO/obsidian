## Mesh들이 다 같이 움직임
#### 발생일 : 2024.12.22.
#### 문제 : Mesh 이동 시 연결된 모든 Mesh가 같이 이동됨
#### 해결법 : 
1. 전체 선택 후 P 눌러서 Detach
2. 전체 선택 후 Object -> Relation -> Single User -> Mesh & Data
3. 전체 선택 후 Object -> Parent -> Clear Parent
4. 위에 Propotional Editing Mode 켜져 있는지 확인
