## 레벨 전환 후 데이터 유지
- 레벨 전환 시 초기화로 인한 노드 진행 멈춤 문제 발생. 해결 법 3가지
1. 게임 인스턴스에 관련 변수 저장
	- Game Instance 파일 생성 후 원하는 변수 생성
	- Project Setting에서 해당 Game Instance로 설정.
	- {Get Game Instance -> Cast To 만든 파일명}으로 해당 Instance 불러온 후 원하는 변수 저장/호출
2. Stream Level 사용
	- Stream Level이란? 다수의 Level들을 한 Level에 몰아넣은 후 필요 시에만 생성하도록 하는 방법
	- Load/Unload Stream Level 노드를 통해 생성/삭제
	- Window\Levels에서 Persistent Level에 Add Existing을 통해 Level들을 추가
3. Level Blueprint 사용(내가 사용한 방법!)
	- Open Level 이후의 노드 동작을 그 Level의 Level Blueprint의 Begin Play로 이동
	- 예시) 레벨 생성 시 전환 트랜지션 효과를 재생 후 Widget의 LPImage및 애니메이션 재생(Cast To WBP_Mainmenu 사용에 주목)
