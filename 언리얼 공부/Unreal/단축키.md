## 1. 단축키 참고사항
---
- LMB : 마우스 좌 클릭
- MMB : 마우스 휠 클릭
- MW : 마우스 휠 스크롤
- RMB : 마우스 우 클릭
- A + B : A와 B를 같이 누른
## 2. 레벨 에디터
---
#### 2.1 에디터

|       단축키        | 설명                       |
| :--------------: | ------------------------ |
|        G         | 게임뷰 토글                   |
|        T         | 반투명 선택 토글                |
|        ~         | 콘솔 입력                    |
|     Ctrl + N     | 새 레벨 생성                  |
|    Ctrl + 0~9    | 북마크 설정                   |
|       0~9        | 북마크에 저장된 위치와 방향으로 뷰포트 이동 |
|     Ctrl + R     | 뷰포트 실시간 켜기/끄기            |
|    Ctrl + Tap    | 탭 내비게이션                  |
| Ctrl + Spacebar  | 콘텐츠 드로어                  |
|     Ctrl + S     | 저장                       |
| Ctrl + Shift + S | (저장되지 않은) 모든 레벨 및 에셋 저장  |
| Ctrl + Shift + ; | 라이팅 빌드                   |
|   Shift + F11    | 에디터 전체 화면                |
|        F9        | 현재 뷰포트 캡처                |
|       F10        | 사이드바 토글                  |
|       F11        | 뷰포트를 몰입/일반모드 전환          |

## 4. 블루프린트 에디터
---
#### 4.1 기본

|       단축키        | 설명            |
| :--------------: | ------------- |
|     Ctrl + B     | 콘텐츠 브라우저에서 찾기 |
|     Ctrl + S     | 블루프린트 저장      |
|     Ctrl + Y     | 다시하기          |
|     Ctrl + Z     | 되돌리기          |
|     Ctrl + F     | 찾기            |
| Ctrl + Shift + F | 모든 블루프린트에서 찾기 |
|        F7        | 컴파일           |

#### 4.2 에디터 생성

|   단축키   | 설명               |
| :-----: | ---------------- |
| A + LMB | Array Get 노드     |
| B + LMB | Branch 노드        |
|  **C**  | **Comment 박스**   |
| D + LMB | Delay 노드         |
| S + LMB | Sequence 노드      |
| G + LMB | Gate 노드          |
| F + LMB | For-Each loop 노드 |
| M + LMB | Multi-gate 노드    |
| N + LMB | Do N times 노드    |
| O + LMB | Do Once 노드       |
| P + LMB | BeginPlay 노드     |

#### 4.3 에디터 조작

|       단축키       | 설명          |
| :-------------: | ----------- |
|    RMB + 드래그    | 그래프 이동      |
|      Home       | 선택에 맞게 줌 설정 |
| LMB + RMB + 드래그 | 확대/축소       |
|    PageDown     | 자손 그래프로 이동  |
|     PageUp      | 부모 그래프로 이동  |

#### 4.4 내부 동작

|         단축키         | 설명           |
| :-----------------: | ------------ |
|        호환핀에         | Get/Set 자동선택 |
|      그래프에 드래그       | Get/Set (메뉴) |
| **그래프에 Ctrl + 드래그** | **Get 변수**   |
| **그래프에 Alt + 드래그**  | **Set 변수**   |
|   Get/Set 노드에 드래그   | 기존 노드 변경     |
|   내 블루프린트 안에서 드래그   | 카테고리/순서 변경   |
